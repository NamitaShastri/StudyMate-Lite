# StudyMate Lite

A clean and lightweight Chrome extension that works as an AI-powered study companion. It helps you understand, summarize, and practice concepts directly from any webpage.

---

## Key Features

* Explain selected text in simple language
* Generate accurate summaries
* Provide concept hints
* Give quick quizzes for revision
* Works via right‑click context menu
* Powered by Gemini 2.5 Flash

---

## Tech Stack

* Chrome Extension (Manifest v3)
* JavaScript, HTML, CSS
* Google Gemini 2.5 Flash API

---

## Getting Started

### Installation (For Users)

* Download the extension folder
* Open **chrome://extensions** in your browser
* Enable **Developer Mode**
* Click **Load Unpacked**
* Select the project folder

### Setup (For Developers)

* Clone this repository
* Add your Gemini API key to `manifest.json`
* Load the project as an unpacked extension in Chrome
